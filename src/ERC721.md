# ERC-721 Token Fetcher Module

A comprehensive module for interacting with ERC-721 (NFT) tokens on blockchain networks. This module provides a clean API for fetching token information, metadata, ownership, and collection details.

## 🎨 Features

- **ERC-721 Standard Compliance**: Implements all standard ERC-721 read functions
- **Address Validation**: Automatic validation and normalization of Ethereum addresses
- **Token ID Validation**: Handles token IDs of any size (including large numbers)
- **Metadata Caching**: Built-in caching for efficient metadata retrieval
- **ABI Access**: Direct access to function ABIs and signatures
- **Type Safety**: Comprehensive input validation and error handling
- **Network Agnostic**: Works with any EVM-compatible blockchain (Ethereum, Base, Polygon, etc.)

## 📦 Installation

The ERC-721 module is included in the project. No additional dependencies are required.

```javascript
const ERC721Fetcher = require('./src/erc721');
```

## 🚀 Quick Start

```javascript
const ERC721Fetcher = require('./src/erc721');

// Create a fetcher for an NFT contract
const fetcher = new ERC721Fetcher('0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D');

// Get token owner
const owner = await fetcher.getOwner(123);
console.log(`Token #123 is owned by ${owner.owner}`);

// Get token metadata
const metadata = await fetcher.getTokenMetadata(123);
console.log(`Token name: ${metadata.name}`);

// Check token balance of an address
const balance = await fetcher.getBalance('0xabcd...1234');
console.log(`Balance: ${balance.balance} tokens`);
```

## 📖 API Reference

### Constructor

#### `new ERC721Fetcher(contractAddress, rpcUrl?)`

Creates a new ERC-721 fetcher instance.

**Parameters:**
- `contractAddress` (string): The ERC-721 contract address (required)
- `rpcUrl` (string): The RPC endpoint URL (optional)

**Example:**
```javascript
const fetcher = new ERC721Fetcher(
  '0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D',
  'https://mainnet.base.org'
);
```

### Token Information Methods

#### `getBalance(ownerAddress)`

Gets the number of tokens owned by an address.

**Parameters:**
- `ownerAddress` (string): The address to check

**Returns:** Promise<object>
```javascript
{
  owner: '0xabcd...1234',
  balance: '5',
  contract: '0x...',
  method: 'balanceOf',
  abi: {...}
}
```

#### `getOwner(tokenId)`

Gets the owner of a specific token.

**Parameters:**
- `tokenId` (string|number): The token ID

**Returns:** Promise<object>
```javascript
{
  tokenId: '123',
  owner: '0xabcd...1234',
  contract: '0x...',
  method: 'ownerOf',
  abi: {...}
}
```

#### `getTokenURI(tokenId)`

Gets the token URI (metadata location).

**Parameters:**
- `tokenId` (string|number): The token ID

**Returns:** Promise<object>
```javascript
{
  tokenId: '123',
  uri: 'ipfs://...',
  contract: '0x...',
  method: 'tokenURI',
  abi: {...}
}
```

#### `getTokenMetadata(tokenId)`

Fetches and parses token metadata.

**Parameters:**
- `tokenId` (string|number): The token ID

**Returns:** Promise<object>
```javascript
{
  tokenId: '123',
  name: 'Token Name',
  description: 'Token description',
  image: 'ipfs://...',
  attributes: [...],
  contract: '0x...'
}
```

**Note:** Metadata is automatically cached for performance.

#### `getTokenInfo(tokenId)`

Gets comprehensive token information in a single call.

**Parameters:**
- `tokenId` (string|number): The token ID

**Returns:** Promise<object>
```javascript
{
  tokenId: '123',
  owner: '0xabcd...1234',
  uri: 'ipfs://...',
  metadata: {...},
  contract: '0x...'
}
```

### Collection Information Methods

#### `getCollectionName()`

Gets the collection name.

**Returns:** Promise<object>
```javascript
{
  name: 'Bored Ape Yacht Club',
  contract: '0x...',
  method: 'name',
  abi: {...}
}
```

#### `getCollectionSymbol()`

Gets the collection symbol.

**Returns:** Promise<object>
```javascript
{
  symbol: 'BAYC',
  contract: '0x...',
  method: 'symbol',
  abi: {...}
}
```

#### `getTotalSupply()`

Gets the total supply of tokens.

**Returns:** Promise<object>
```javascript
{
  totalSupply: '10000',
  contract: '0x...',
  method: 'totalSupply',
  abi: {...}
}
```

### Verification Methods

#### `verifyOwnership(ownerAddress, tokenId)`

Verifies if an address owns a specific token.

**Parameters:**
- `ownerAddress` (string): The address to verify
- `tokenId` (string|number): The token ID

**Returns:** Promise<object>
```javascript
{
  owner: '0xabcd...1234',
  tokenId: '123',
  isOwner: true,
  contract: '0x...'
}
```

### Utility Methods

#### `validateAddress(address)`

Validates and normalizes an Ethereum address.

**Parameters:**
- `address` (string): The address to validate

**Returns:** string (normalized address)

**Throws:** Error if address is invalid

#### `validateTokenId(tokenId)`

Validates a token ID.

**Parameters:**
- `tokenId` (string|number): The token ID to validate

**Returns:** string (normalized token ID)

**Throws:** Error if token ID is invalid

#### `clearCache()`

Clears the internal metadata cache.

**Returns:** void

### Static Methods

#### `ERC721Fetcher.getABI(functionName)`

Gets the ABI for a specific function.

**Parameters:**
- `functionName` (string): The function name

**Returns:** object|null

**Example:**
```javascript
const abi = ERC721Fetcher.getABI('balanceOf');
console.log(abi.signature); // '0x70a08231'
```

#### `ERC721Fetcher.getAllABIs()`

Gets all ERC-721 standard ABIs.

**Returns:** object

**Example:**
```javascript
const abis = ERC721Fetcher.getAllABIs();
console.log(Object.keys(abis)); // ['balanceOf', 'ownerOf', 'tokenURI', ...]
```

## 🔧 Usage Examples

### Example 1: Check NFT Ownership

```javascript
const fetcher = new ERC721Fetcher('0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D');

// Check if an address owns a specific token
const result = await fetcher.verifyOwnership(
  '0xabcdefabcdef1234567890abcdefabcdef123456',
  1234
);

if (result.isOwner) {
  console.log('Address owns this token!');
} else {
  console.log('Address does not own this token.');
}
```

### Example 2: Get All Token Info

```javascript
const fetcher = new ERC721Fetcher('0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D');

// Get comprehensive token information
const tokenInfo = await fetcher.getTokenInfo(42);

console.log(`Token #${tokenInfo.tokenId}`);
console.log(`Owner: ${tokenInfo.owner}`);
console.log(`Metadata:`, tokenInfo.metadata);
```

### Example 3: Get Collection Stats

```javascript
const fetcher = new ERC721Fetcher('0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D');

const [name, symbol, supply] = await Promise.all([
  fetcher.getCollectionName(),
  fetcher.getCollectionSymbol(),
  fetcher.getTotalSupply()
]);

console.log(`${name.name} (${symbol.symbol})`);
console.log(`Total Supply: ${supply.totalSupply}`);
```

### Example 4: Working with Function ABIs

```javascript
// Get the ABI for balanceOf function
const balanceOfABI = ERC721Fetcher.getABI('balanceOf');

console.log('Function:', balanceOfABI.name);
console.log('Signature:', balanceOfABI.signature);
console.log('Inputs:', balanceOfABI.inputs);
console.log('Outputs:', balanceOfABI.outputs);

// Output:
// Function: balanceOf
// Signature: 0x70a08231
// Inputs: [{ name: 'owner', type: 'address' }]
// Outputs: [{ name: 'balance', type: 'uint256' }]
```

### Example 5: Handling Multiple Tokens

```javascript
const fetcher = new ERC721Fetcher('0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D');

// Fetch metadata for multiple tokens in parallel
const tokenIds = [1, 2, 3, 4, 5];
const metadataPromises = tokenIds.map(id => fetcher.getTokenMetadata(id));
const allMetadata = await Promise.all(metadataPromises);

allMetadata.forEach(metadata => {
  console.log(`Token #${metadata.tokenId}: ${metadata.name}`);
});
```
