## Overview

The ERC20Fetcher module provides utilities for interacting with ERC-20 fungible tokens on any EVM-compatible blockchain.

## Installation

```bash
npm install big-world-bigger-ideas
```

## Basic Usage

```javascript
const { ERC20Fetcher } = require('big-world-bigger-ideas');

const fetcher = new ERC20Fetcher('0xTokenAddress...');
const balance = await fetcher.balanceOf('0xAddress...');
console.log('Token balance:', balance);
```
