## Overview

The ERC721Fetcher class provides methods to interact with ERC-721 NFT contracts on any EVM-compatible blockchain. It supports fetching owner information, token URIs, metadata, and more.

## Installation

```bash
npm install big-world-bigger-ideas
```

## Basic Usage

```javascript
const { ERC721Fetcher } = require('big-world-bigger-ideas');

// Initialize with contract address
const nftFetcher = new ERC721Fetcher('0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D');

// Get NFT owner
const owner = await nftFetcher.getOwner('1');
console.log('Owner:', owner);
```

## Constructor

### `new ERC721Fetcher(contractAddress, rpcUrl?)`

Creates a new ERC721Fetcher instance.

**Parameters:**

<ParamField path="contractAddress" type="string" required>
  The Ethereum address of the ERC-721 contract (42 or 40 characters)
</ParamField>

<ParamField path="rpcUrl" type="string" optional default="https://ethereum.publicnode.com">
  The RPC endpoint URL for the blockchain
</ParamField>

**Example:**

```javascript
// Using default Ethereum RPC
const fetcher = new ERC721Fetcher('0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D');

// Using custom RPC (e.g., Polygon)
const polygonFetcher = new ERC721Fetcher(
  '0x...',
  'https://polygon-rpc.com'
);
```

## Methods

### `getOwner(tokenId)`

Get the owner of a specific token.

**Parameters:**

<ParamField path="tokenId" type="string | number" required>
  The token ID to query
</ParamField>

**Returns:** `Promise<string>` - The owner's Ethereum address

**Example:**

```javascript
const owner = await fetcher.getOwner('1');
console.log('Token owner:', owner);
```

### `getTokenURI(tokenId)`

Get the token URI (metadata location) for a token.

**Parameters:**

<ParamField path="tokenId" type="string | number" required>
  The token ID to query
</ParamField>

**Returns:** `Promise<string>` - The token URI (usually an IPFS or HTTP URL)

**Example:**

```javascript
const uri = await fetcher.getTokenURI('1');
console.log('Token URI:', uri);
```

### `getName()`

Get the collection name.

**Returns:** `Promise<string>` - The contract name

**Example:**

```javascript
const name = await fetcher.getName();
console.log('Collection name:', name);
```

### `getSymbol()`

Get the collection symbol.

**Returns:** `Promise<string>` - The contract symbol

**Example:**

```javascript
const symbol = await fetcher.getSymbol();
console.log('Collection symbol:', symbol);
```

### `getTotalSupply()`

Get the total supply of tokens in the collection.

**Returns:** `Promise<string>` - The total supply as a string

**Example:**

```javascript
const supply = await fetcher.getTotalSupply();
console.log('Total supply:', supply);
```

### `getBalanceOf(address)`

Get the token balance of an address.

**Parameters:**

<ParamField path="address" type="string" required>
  The Ethereum address to query
</ParamField>

**Returns:** `Promise<string>` - The number of tokens owned by the address

**Example:**

```javascript
const balance = await fetcher.getBalanceOf('0x1234...');
console.log('Balance:', balance);
```

## Supported Chains

The ERC721Fetcher works with any EVM-compatible blockchain:

- **Ethereum** - Mainnet and testnets
- **Polygon** - MATIC network
- **Binance Smart Chain** - BSC
- **Avalanche** - C-Chain
- **Arbitrum** - Layer 2
- **Optimism** - Layer 2
- **Base** - Coinbase L2
- And more...

Just provide the appropriate RPC URL for your target chain.

## Error Handling

The fetcher validates inputs and throws descriptive errors:

```javascript
try {
  const owner = await fetcher.getOwner('invalid-token-id');
} catch (error) {
  console.error('Error:', error.message);
  // Handle error appropriately
}
```

**Common errors:**

- Invalid contract address format
- Invalid token ID
- Token doesn't exist
- Network connection issues
- Contract doesn't implement ERC-721

## Advanced Usage

### Fetching Metadata

```javascript
// Get token URI
const uri = await fetcher.getTokenURI('1');

// Fetch metadata from URI
const response = await fetch(uri.replace('ipfs://', 'https://ipfs.io/ipfs/'));
const metadata = await response.json();

console.log('NFT Metadata:', metadata);
```

### Batch Operations

```javascript
// Fetch multiple token owners
const tokenIds = ['1', '2', '3', '4', '5'];
const owners = await Promise.all(
  tokenIds.map(id => fetcher.getOwner(id))
);

console.log('Owners:', owners);
```

### Working with Different Chains

```javascript
// Ethereum mainnet
const ethFetcher = new ERC721Fetcher(
  '0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D',
  'https://ethereum.publicnode.com'
);

// Polygon
const polygonFetcher = new ERC721Fetcher(
  '0x...',
  'https://polygon-rpc.com'
);

// Base
const baseFetcher = new ERC721Fetcher(
  '0x...',
  'https://mainnet.base.org'
);
```

## Caching

The ERC721Fetcher implements caching with a 60-second timeout to reduce RPC calls:

```javascript
// First call - fetches from blockchain
const owner1 = await fetcher.getOwner('1');

// Second call within 60s - returns cached result
const owner2 = await fetcher.getOwner('1');

// Clear cache manually if needed
fetcher.clearCache();
```
