## Overview

The BitcoinMiningFetcher provides access to Bitcoin mining data including hash rate, difficulty adjustments, and network statistics from mempool.space API.

## Installation

```bash
npm install big-world-bigger-ideas
```

## Basic Usage

```javascript
const { BitcoinMiningFetcher } = require('big-world-bigger-ideas');

const fetcher = new BitcoinMiningFetcher();
const hashRate = await fetcher.getHashRate('1w');
console.log('Hash rate:', hashRate);
```
