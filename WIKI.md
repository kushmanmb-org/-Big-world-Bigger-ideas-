## 🔄 Updating the Wiki

### For Maintainers

When updating the wiki:

1. **Edit source files** in the `/wiki` directory
2. **Test locally** to ensure markdown renders correctly
3. **Commit changes** to the repository
4. **Update wiki** via GitHub web interface or git clone:

```bash
# Clone wiki repository
git clone https://github.com/kushmanmb-org/-Big-world-Bigger-ideas-.wiki.git

# Copy updated files
cp wiki/*.md -Big-world-Bigger-ideas-.wiki/

# Commit and push
cd -Big-world-Bigger-ideas-.wiki
git add .
git commit -m "Update wiki documentation"
git push origin master
```

### For Contributors

To contribute to the wiki:

1. **Edit files** in the `/wiki` directory
2. **Submit a pull request** with your changes
3. **Maintainers will review** and update the live wiki

See [Contributing Guide](https://github.com/kushmanmb-org/-Big-world-Bigger-ideas-/wiki/Contributing) for details.
