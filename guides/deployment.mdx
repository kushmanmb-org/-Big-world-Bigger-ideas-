## Overview

This guide covers deployment strategies for the Big World Bigger Ideas platform.

## Quick Start

The platform is automatically deployed to GitHub Pages when changes are pushed to the `main` branch.

- **Live Site:** https://kushmanmb.org
- **Deployment Workflow:** `.github/workflows/deploy.yml`
- **Platform:** GitHub Pages with custom domain

## Deployment Checklist

Before deploying, review the [Deployment Checklist](https://github.com/kushmanmb-org/-Big-world-Bigger-ideas-/blob/main/DEPLOYMENT-CHECKLIST.md) to ensure all requirements are met.

## Manual Deployment

To manually trigger a deployment:

1. Go to the [Actions tab](https://github.com/kushmanmb-org/-Big-world-Bigger-ideas-/actions/workflows/deploy.yml)
2. Click "Run workflow"
3. Select the `main` branch
4. Click "Run workflow"

## Verification

After deployment:

1. Visit https://kushmanmb.org
2. Check that all pages load correctly
3. Verify documentation is accessible
4. Test feature flags and JavaScript modules
